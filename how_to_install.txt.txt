Ce document contient les commandes a exécuter dans le terminal afin d'éxecuter le code fourni 




1 - PyTorch et torchvision : 

# Installation de PyTorch avec CPU seulement
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu

# OU, pour GPU (CUDA) si tu as une carte NVIDIA compatible
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121


2 - Matplotlib : 

pip install matplotlib


3 - NumPy : 

pip install numpy


4 - tqdm (barre de progression) 

pip install tqdm


5 - scikit-image 

pip install scikit-image
